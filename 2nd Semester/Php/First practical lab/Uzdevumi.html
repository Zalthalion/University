PART 1: TEST YOUR LOCAL WEB SERVER

  1. Start WampServer
  2. Check WampServer icon on the right -- it is ready when it is green.

    If it is not green, Left-Click -> Restart All Services.
  3. Left-Click, see Apache, PHP, and MySQL menus.
  4. Left-Click -> www directory: This is the directory where we put our PHP or HTML files.

    Create a directory with your name and put your PHP files there during the lab.

    Keep open this directory during the lab.
  5. Left-Click -> Localhost: This is how we can access our local server, usually http://localhost/.

    You should see the main page: "index.php" under the "www directory".

    Please never delete or modify "index.php" under the "www directory".

    Please use the directory with your name for your work.
  6. http://localhost/: There are certain tools you may use. We will use them in the upcoming labs.
  7. Go to your directory with your name. Create a PHP file named "first.php"
  8. Edit "first.php" (notepad, notepad++, eclipse, or your favorite editor). 
    <?php
echo "hello world";
?>
  9. Call your file from the server by entering the address bar of your browser: http://localhost/your_name/first.php

    This is how we create, edit, and then test our php files.
